AllOverIt.Aws.Cdk.AppSync - v0.8.0
  * Breaking changes, now enforcing the use of methods to describe a Graphql schema. Allowing the use of properties

  * Added/updated exception messages for invalid declarations such as nullable returns types and method arguments,
    missing datasource atttributes, and the use of properties.

  * Guard.WhenNotEmpty() and Guard.WhenNotNullOrEmpty() now enforce a check that ensures the Enumerable is an array

  * Added Formatter.FormatJsonString() utility method. Beautifies a JSON formatted string. No validation of content.
